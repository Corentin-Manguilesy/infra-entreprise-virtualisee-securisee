# Infrastructure d’entreprise virtualisée sécurisée

## Objectif
Concevoir et déployer une infrastructure d’entreprise simulée avec segmentation réseau et services centralisés.

## Architecture

- OPNsense (pare-feu)
- Active Directory
- Serveur Linux Debian
- Apache / PHP
- MariaDB

Architecture 3-tiers :

Client → Apache/PHP → MariaDB

## Fonctionnalités

- Pare-feu OPNsense avec règles réseau
- Active Directory et DNS interne
- Application web intranet
- Base de données MariaDB
- Segmentation réseau

## Technologies utilisées

- Windows Server 2022
- Debian Linux
- OPNsense
- Apache
- MariaDB
- Php
